OctoPrint - это веб-интерфейс для вашего 3D-принтера.

Поддерживаемые языки:
 Английский
 Нидерландский
 Норвежский

Функции:
- Подключение/отключение
- Установка/получение температуры инструментов
- Установка/получение температуры платформы
- Установка/получение температуры камеры
- Получение состояний
- Получение данных с датчиков
- Получение времени
- Получение камеры (снимок)
