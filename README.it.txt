OctoPrint è un'interfaccia web per la tua stampante 3D.

Lingue supportate:
 Inglese
 Olandese
 Norvegese

Funzionalità:
- Connetti/disconnetti
- Imposta/ottieni temperature degli strumenti
- Imposta/ottieni temperature del piano di stampa
- Imposta/ottieni temperature della camera
- Ottieni stati
- Ottieni sensori
- Ottieni tempi
- Ottieni fotocamera (istantanea)
