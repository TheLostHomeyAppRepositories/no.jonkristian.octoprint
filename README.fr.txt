OctoPrint est une interface web pour votre imprimante 3D.

Langues prises en charge :
 Anglais
 Néerlandais
 Norvégien

Fonctionnalités :
- Connecter/déconnecter
- Régler/Obtenir les températures des outils
- Régler/Obtenir les températures du plateau
- Régler/Obtenir les températures de la chambre
- Obtenir les états
- Obtenir les capteurs
- Obtenir les temps
- Obtenir la caméra (instantané)
